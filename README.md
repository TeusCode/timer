Cronomêtro criado para estudo prático de lógica de contagem de tempo e manipulação de interface, foi usado no projeto:
- HTML5 e CSS3: para criar um visual limpo, o fundo e os botões,
- JavaScript: Para fazer toda a lógica, tempo rodar, pausar e resetar.
O designer foi inspierado em conceitos de design aberto da comunidade tech.
